## debugging tests

# node-cli
1. run: mocha debug tests/tests.js -R spec

# node-inspector
1. run: mocha --debug-brk tests/test.js
2. run: node-inspector
3. navigate to: http://127.0.0.1:8080/debug?port=5858

## debugging grunt itself

grunt test -v --debug
or
grunt -v -d 1

## drop debugger and console.log lines
If you're using UglifyJS2 then you most likely don't need this package at all. You can just use the drop_debugger and drop_console to achieve the same effect.
